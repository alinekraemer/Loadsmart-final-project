API

Our API allows Shippers to:

  • Post Loads
  • Get Available and Accepted Loads Information

Our API allows Carriers to:

  • Accept, Reject and Drop Loads
  • Get Available, Accepted and Rejected Loads Information

  • BASH 

import requests
url = "http://loadsmart-aceleradev.herokuapp.com/api/shipper/load/post/"

load = {
          "pickup_date": "2019-09-13",
          "ref": 3550,
          "origin_city": "Houston",
          "destination_city": "San Diego",
          "price": 1000
        }

headers = {'Authorization': 'Token 09211697a0acc58e598f18f13d6b1cd73ebd91e6'}

response = requests.post(url, json=load, headers=headers)
print(response.content)
