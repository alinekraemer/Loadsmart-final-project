Loadsmart API Documentation¶

Authentication¶

  1. Register on our website:

    https://loadsmart-aceleradev.herokuapp.com/

  2. Get your token to use Loadsmart API using your credentials (email and password):

    Request: POST /api/token/
    Parameters: email, password
    Returns: Your token to use the API

    • Example:

      curl -d "email=johndoe@test.com&password=1234" \
        -X POST https://loadsmart-aceleradev.herokuapp.com/api/token/

    • Expected Success Response:

      HTTP Status Code: 200

{"token":"a58d372f58af11ac458f3695bb8504ca8e462af4"}

    • Expected Fail Response:

      HTTP Status Code: 404

{"error":"Invalid Credentials"}

or

HTTP Status Code: 400

{"error": "Please provide both email and password"}

For Shippers¶

List Available Loads¶

This method allows users to get a list of their available loads.

It returns the following information for each load: ID, Pick up date, Reference, Origin city, Destination city, Price, Status, Shipper’s name and Carrier’s name.

Request: GET /api/shipper/loads/available/
Parameters: Authentication token
Returns: List of available loads in JSON format

  • Example:

    curl -H "Authorization: Token 09211697a0acc58e598f18f13d6b1cd73ebd91e6" \
        -X GET http://loadsmart-aceleradev.herokuapp.com/api/shipper/loads/available/

  • Expected Success Response:

    HTTP Status Code: 200

{
  "count": 2,
  "next": null,
  "previous": null,
  "results": [
         { "id": 1,
           "pickup_date": "2019-05-01",
           "ref": "3345",
           "origin_city": "Miami Beach, FL, USA",
           "destination_city": "Orlando, FL, USA",
           "price": 350,
           "status": "available",
           "shipper": "Shipper1 Doe",
           "carrier": null
         },
         { "id": 3,
           "pickup_date": "2019-05-04",
           "ref": "9090",
           "origin_city": "Dallas, TX, USA",
           "destination_city": "Los Angeles, CA, USA",
           "price": 2200,
           "status": "available",
           "shipper": "Shipper1 Doe",
           "carrier": null
         }
  ]
}

  • Expected Fail Response:

    HTTP Status Code: 401

{"detail": "Authentication credentials were not provided."}

or

HTTP Status Code: 401

{"detail": "Invalid token."}

Post a Load¶

This method allows users to post a new load.

Request:

POST /shipper/load/post/
Parameters:
  • Authentication token
  • pickup_date: string (YYYY-MM-DD)
  • ref: string
  • origin_city: string
  • destination_city: string
  • price: float
Returns:

Message informing that the load was posted successfully

  • Example:

    curl -H "Authorization: Token 09211697a0acc58e598f18f13d6b1cd73ebd91e6" \
        -H "Content-Type: application/json" \
        -d "{"pickup_date":"2019-09-13", "ref":"3550", "origin_city":"Houston", "destination_city":"San Diego", "price":1000}" \
        -X POST http://loadsmart-aceleradev.herokuapp.com/api/shipper/load/post/

  • Expected Success Response:

    HTTP Status Code: 200

{"success": "Load posted successfully"}

  • Expected Fail Response:

    HTTP Status Code: 401

{"detail": "Authentication credentials were not provided."}

or

HTTP Status Code: 401

{"detail": "Invalid token."}

Accept a Load¶

This method allows users to accept a load.

Only available loads can be accepted. In order to see which loads are available and their IDs, see List Available Loads.

Request:

PUT /api/carrier/load/accept/<int:pk_load>/
Parameters:
  • Authentication token
  • load_id: integer
Returns:

Message informing that the load was accepted successfully

  • Example:

    curl -H "Authorization: Token a58d372f58af11ac458f3695bb8504ca8e462af4" \
        -X PUT http://loadsmart-aceleradev.herokuapp.com/api/carrier/load/accept/3/

  • Expected Success Response:

    HTTP Status Code: 200

{"success":"Load accepted successfully"}

  • Expected Fail Response:

    HTTP Status Code: 401

{"detail": "Authentication credentials were not provided."}

or

HTTP Status Code: 401

{"detail": "Invalid token."}

or

HTTP Status Code: 404

{"error": "Load not found or Load not available"}
